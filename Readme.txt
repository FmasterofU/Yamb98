Yamb98 sadrzi sledece fajlove:

	Yamb98.exe    - izvrsna datoteka
	VerdanYU.ttf  - YU font "Verdana"
	Readme.txt    - ovaj tekst


Instalacija:

  Napravite precicu do datoteke Yamb98.exe i instalacija je gotova.

  Ako zelite da vidite nasa (YU) slova iskopirajte font VerdanYU.ttf
u c:\windows\fonts\


Potreban hardver:

486 /133 MHz (ili brzi)
8 Mb Ram
2 Mb prostora na disku


Potreban softver:

	Windows 95/98
	datoteka: "Msvbvm50.dll" u c:\windows\system (ova datoteka
		je sastavni deo Windowsa 98, a koriste je svi pro-
		grami napisani u Visual Basicu 5). Nije ukljucena
		uz program zbog velicine (1,29 Mb). Ako je nemate
		javite se, poslacu Vam je.

Deinstalacija:

  Obrisite gore navedene datoteke :)
